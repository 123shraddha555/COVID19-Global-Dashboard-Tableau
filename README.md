# COVID-19 Global Dashboard

Interactive COVID-19 Global Dashboard built using Tableau.

---

## 📌 Project Overview

This project is an interactive Tableau dashboard developed to visualize global COVID-19 statistics. It provides insights into confirmed cases, deaths, recoveries, active cases, and population across countries and continents.

---

## 🛠️ Tools Used

- Tableau
- Excel
- Data Visualization

---

## 📊 Dashboard Features

- Total Cases KPI
- Total Deaths KPI
- Total Recovered KPI
- Total Active Cases KPI
- Total Population KPI
- Filled World Map
- Top 10 Countries by Cases
- Cases by Continent
- Active vs Recovered Analysis
- Death Rate Analysis
- Country & Continent Filters
